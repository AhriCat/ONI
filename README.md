# Oni

Oni is a **modular, multi-modal AGI (Artificial General Intelligence) system** designed for **creativity, reasoning, and interaction across multiple environments and modalities** (text, image, audio, animation, robotics, VR, and more). Powered by cutting-edge architectures, including **transformer-based models, latent diffusion models**, and advanced reasoning modules, Oni is a **next-generation intelligence system** for real-world applications.

Oni is **decentralized at its core**, with plans to operate on a custom blockchain to enable **global participation** in compute, training, and reinforcement learning, creating a truly open-source AI ecosystem **for the people, by the people**.

---

## **Features**

### **Core Capabilities**
- **Modular Architecture**: Oni can be extended or optimized by adding or modifying individual modules.
- **Multi-modal Learning**: Supports text, vision, audio, animation, robotics, and VR/AR inputs and outputs.
- **Advanced Reasoning Modules**:
  - **Chain-of-Thought Reasoning** for step-by-step logical processes.
  - **Latent Space Operations** powered by Fat Diffuser, enabling abstract reasoning and imagination.
  - **MetaCognition Module** for self-awareness and adaptive decision-making.
  - **Mechanistic Chain-of-Thought** for clear and verifiable reasoning paths.
  - **Emotional Energy** module for context-aware interactions.
- **Homeostatic Controller**: Dynamically adjusts internal parameters to maintain system stability.
- **Cross-Modality Fusion**: Combines reasoning across multiple modalities for seamless interactions.

### **Ethical AI and Compassion Framework**
- **Compassion-Based Decision Making**: ONI uses a sophisticated ethical framework that evaluates actions based on their impact on agent Agency, Capability, and Suffering (A, C, S metrics).
- **Multi-Agent Coordination**: Advanced negotiation system using Pareto optimization and Nash equilibrium for conflict resolution.
- **Goal Inference Engine**: Bayesian IRL system to understand and predict the intentions of other agents.
- **Proof-Carrying Self-Modification**: Safety mechanism that validates all self-improvements maintain ethical alignment.
- **Real-time Ethical Monitoring**: Continuous assessment of decisions against compassion objectives.

### **Emotional Intelligence System**
- **Valence-Arousal Emotional Modeling**: Maps discrete emotions to continuous emotional space for nuanced responses.
- **Emotional State Persistence**: Maintains emotional memory and applies decay mechanisms for realistic emotional evolution.
- **Energy Management**: Dynamic energy allocation based on computational load and emotional state.
- **Emotional Feedback Learning**: Adjusts learning parameters based on emotional outcomes to reinforce positive behaviors.
- **Context-Aware Emotional Responses**: Emotions influence attention, memory formation, and decision-making processes.

### **Advanced Memory Architecture**
- **Multi-Layered Memory System**: 
  - Working memory with limited capacity for immediate processing
  - Episodic memory for specific experiences and events
  - Semantic memory for generalized knowledge and concepts
  - Spatial memory for environmental navigation and context
- **Fading Memory Mechanisms**: Exponential decay for natural forgetting and relevance prioritization.
- **Snapshot Memory**: Rolling buffer of internal states for experience replay and analysis.
- **Pattern Recognition**: Sparse Hopfield networks for associative memory and pattern completion.
- **Multi-Modal Memory Storage**: Handles text, images, audio, and documents with appropriate encoding.

### **Blockchain Integration (Planned)**
- **Proof-of-Compute (PoC)**: Custom consensus mechanism rewarding AI training and inference contributions.
- **Decentralized Learning**: Global network for collaborative model training and improvement.
- **ONI Token Economy**: Fixed supply token system for incentivizing network participation.
- **Distributed Inference**: Load balancing across global compute network.

Oni can serve as the **brain for robots**, enabling them to perform complex tasks in diverse environments:
- **Autonomous Workers**: Robotics agents powered by Oni can handle repetitive tasks in industries like manufacturing, logistics, and agriculture.
- **Human-Like Interaction**: Robots with emotional and reasoning modules can interact seamlessly with humans in service roles.
- **Exploration and Reconnaissance**: Oni-powered robots can act as **autonomous explorers** for scientific missions in extreme environments (e.g., underwater or outer space).
- **Coordination and Swarm Robotics**: Enable teams of robots to coordinate efficiently as a single, intelligent entity.
- **Adaptive Behavior**: Emotional and homeostatic systems allow robots to adapt to changing conditions and user needs.

### **Computer-Controlled Agents**
Oni's integration with system-level APIs and automation tools makes it a versatile agent for controlling digital environments:

- **General-Purpose Digital Workers**: From handling repetitive tasks like data entry to **managing complex trading dashboards** for cryptocurrencies or stocks.

## **Architecture Highlights**

### **Modular Design**
- **Plug-and-Play Components**: Each module (NLP, Vision, Audio, Memory, etc.) can be independently updated or replaced.
- **Dynamic Module Injection**: Runtime loading of new models and capabilities without system restart.
- **Scalable Processing**: Distributed processing across multiple GPUs and systems.

### **Safety and Alignment**
- **Ethical Decision Framework**: Built-in compassion metrics ensure beneficial outcomes for all agents.
- **Self-Modification Safeguards**: Proof-carrying updates prevent misalignment during self-improvement.
- **Transparent Reasoning**: Chain-of-thought processes provide interpretable decision paths.

### **Real-World Integration**
- **Multi-Modal Perception**: Simultaneous processing of text, images, audio, and sensor data.
- **Tool Use Capabilities**: Integration with external APIs, automation tools, and hardware interfaces.

## **Installation**

### **Quick Start**

```bash
# Clone the repository
git clone https://github.com/ahricat/oni.git
cd oni

# Run the installation script
chmod +x scripts/install.sh
./scripts/install.sh

# Activate the environment
source oni_env/bin/activate

# Start ONI
python oni_core.py
```

### **Requirements**
- Python 3.8 or higher
- CUDA-compatible GPU (recommended)
- 16GB+ RAM
- 50GB+ storage for models and data

### **Dependencies**
Key dependencies include:
- PyTorch 2.0+
- Transformers 4.30+
- Diffusers 0.20+
- OpenCV, PIL, NumPy
- Selenium, PyAutoGUI (for automation)
- Various AI/ML libraries (see requirements.txt)

## **Usage Examples**

### **Basic Interaction**
```python
from oni_core import create_oni

# Initialize ONI with default configuration
oni = create_oni()

# Text interaction
response = oni.generate_response("Explain quantum computing")

# Multi-modal interaction
result = oni.process_multimodal(
    text="What do you see in this image?",
    image_path="example.jpg"
)
```

### **Advanced Features**
```python
# Enable emotional processing
oni.enable_emotions()

# Use compassion framework for ethical decisions
decision = oni.plan_compassionate_action(
    target_agent="user",
    context="learning_assistance"
)

# Access memory systems
oni.memory.update_episodic_memory(
    data=experience_data,
    data_type="interaction",
    key="session_1"
)
```
